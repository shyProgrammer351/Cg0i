A양, 가위, 바위, 보 중 하나를 선택하세요: 바위
바위를 냈습니다.
B군, 가위, 바위, 보 중 하나를 선택하세요: 가위
가위를 냈습니다.
A양이 이겼습니다. 


A양, 가위, 바위, 보 중 하나를 선택하세요: 보
보를 냈습니다.
B군, 가위, 바위, 보 중 하나를 선택하세요: 가위
가위를 냈습니다.
B군이 이겼습니다. 


A양, 가위, 바위, 보 중 하나를 선택하세요: 바위
바위를 냈습니다.
B군, 가위, 바위, 보 중 하나를 선택하세요: 바위
바위를 냈습니다.
A양과 B군이 비겼습니다.

알고리즘

프로그램 시작 
   A양 가위바위보 선택 
   B군 가위바위보 선택 
   가위바위보 승패 출력 
      먼저 A양과 B군이 같으면 비김 
      A양이 가위이고 B군이 보를 낸 경우 A양 승리 
      A양이 바위이고 B군이 가위를 낸 경우 A양 승리 
      A양이 보이고 B군이 바위를 낸 경우 A양 승리 
      나머지 경우 B군 승리 
프로그램 종료

프로그램 코드 

	// 파일명 : ./Chapter10/RockPaperScissors.c

	#include <stdio.h>

	#include <string.h>

	 

	// 프로그램 시작 

1	int main() {

		// A양 가위바위보 선택 

		printf( "A양, 가위, 바위, 보 중 하나를 선택하세요: " );

		char playerA[10] = "";

		scanf( "%s", playerA );

2		printf( "%s를 냈습니다.\n", playerA );

	 

		// B군 가위바위보 선택 

		printf( "B군, 가위, 바위, 보 중 하나를 선택하세요: " );

		char playerB[10] = "";

		scanf( "%s", playerB );

3		printf( "%s를 냈습니다.\n", playerB );

	 

		// 가위바위보 승패 출력 

		// 먼저 A양과 B군이 같으면 비김  

4		if ( !strcmp( playerA, playerB ) ) {

5			printf( "\nA양과 B군이 비겼습니다.\n" );

		}

		// A양이 가위이고 B군이 보를 낸 경우 A양 승리 

6		else if ( !strcmp( playerA, "가위" ) && !strcmp( playerB, "보" ) )	 {

7			printf( "\nA양이 이겼습니다.\n" );

		}

		// A양이 바위이고 B군이 가위를 낸 경우 A양 승리 

8		else if ( !strcmp( playerA, "바위" ) && !strcmp( playerB, "가위" ) ) {

9			printf( "\nA양이 이겼습니다.\n" );

		}

		// A양이 보이고 B군이 바위를 낸 경우 A양 승리 

10		else if ( !strcmp( playerA, "보" ) && !strcmp( playerB, "바위" ) ) {

11			printf( "\nA양이 이겼습니다.\n" );

		}

		// 나머지 경우 B군 승리  

12		else {

13			printf( "\nB군이 이겼습니다.\n" );

		}

	 

		// 프로그램 종료 

14		return 0;

	}
