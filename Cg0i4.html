알고리즘

 프로그램 시작 
    먼저 카드 게임 현황 출력 
       이 때 같은 색 카드를 선택하면 카드 번호 대신 카드 색깔 출력 
    게임 종료 확인 
    게임 종료 아니면 카드 두 장 새로 선택받기 
 프로그램 종료 
 
 카드 게임 현황 출력 함수 
    사용자가 선택한 카드이거나 이미 색깔을 맞춘 카드는 색깔 출력 
    나머지는 번호 출력 
 
 게임 종료 확인 함수 
    11 ~ 44 사이의 카드가 남아 있으면 게임 계속 
    아니면 게임 종료
