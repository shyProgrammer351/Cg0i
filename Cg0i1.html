C 프로그래밍

문제

사용자가 입력한 두 이름이 같은 성인지 다른 성인지 판단하는 문제입니다 이를 해결하는 다음 프로그램을 해석하세요 
A양, 이름을 입력하세요: 허난설헌
B군, 이름을 입력하세요: 허균
허난설헌과 허균은 성이 같습니다.


A양, 이름을 입력하세요: 신사임당
B군, 이름을 입력하세요: 왕건
신사임당과 왕건은 성이 다릅니다.


A양, 이름을 입력하세요: 김만덕
B군, 이름을 입력하세요: 김유신
김만덕과 김유신은 성이 같습니다.

알고리즘

프로그램시작 
   이름 2개를 입력 
   이름에서 첫 음절(2byte)을 비교하기 
      첫 음절이 같으면 성이 같습니다 출력 
      첫 음절이 다르면 성이 다릅니다 출력 
프로그램 종료

프로그램 코드 

	// 파일명 : ./Chapter05/FamilyName.c 

 	#include <stdio.h> 

 	#include <string.h> 				 

	 

	// 프로그램시작 

1	int main() 

	{ 

		// 한 칸당 최대 10byte 문자열을 저장할 수 있는 배열을 선언  // [주석포함]  

		char nameA[10] = { '\0' };					 

2		char nameB[10] = { '\0' };					 

		 

		// 이름 2개를 입력  

		printf( "A양, 이름을 입력하세요: " ); 

		// 허    난    설    헌   \0을 입력한다고 가정 

3		scanf( "%s", nameA ); 

		printf( "B군, 이름을 입력하세요: " ); 

		// 허    균   \0을 입력한다고 가정 

4		scanf( "%s", nameB ); 

	 

		// 이름에서 첫 음절(2byte)을 비교하기 

		// 첫 음절이 같으면 성이 같습니다 출력 

5		if ( !strncmp( nameA, nameB, 2 ) )				 

		{ 

6			printf( "%s과 %s은 성이 같습니다. ", nameA, nameB ); 

		} 

		// 첫 음절이 다르면 성이 다릅니다 출력 

7		else 

		{ 

8			printf( "%s과 %s은 성이 다릅니다. ", nameA, nameB ); 

		} 

		 

		// 프로그램 종료 

9		return 0; 

	}
